Guía de Demostración Hackless

1. Inicio de sesión

Ir a login
Pantalla de acceso para usuarios y administradores.

2. Registro de usuario

Ir a registro
Permite crear nuevos usuarios para pruebas o uso real.

3. Solicitud de demo

Ir a solicitud de demo
Formulario para que cualquier usuario solicite una demo del sistema.

4. Funciones de administrador

  • Login admin
    Inicia sesión como administrador para acceder a la gestión avanzada.
  • Ver solicitudes de demo
    Visualiza, filtra y exporta las solicitudes recibidas.
  • Exportar a Excel
    Haz clic en el botón "Exportar a Excel" en la pantalla de solicitudes para descargar el archivo CSV.

5. Recuperación de contraseña y 2FA

  • Recuperar contraseña admin
    Solicita un enlace de recuperación para administradores.
  • Restablecer contraseña
    Completa el formulario con el token recibido por email/consola.
  • Activar 2FA
    Desde la vista de admin, accede a la opción de seguridad para activar/desactivar 2FA.

6. Gestión de documentos

  • Subir y ver documentos
    Carga, visualiza y gestiona documentos asociados a usuarios.
  • Nómina de empleados
    Vista completa del personal con estatus de documentación y fechas de vencimiento.

7. Capacitación y módulos formativos

Ir a capacitación
Visualiza y marca como completados los módulos de formación en ciberseguridad.

Instrucciones detalladas para usuarios y administradores.
